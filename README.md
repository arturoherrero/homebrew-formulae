# Homebrew Formulae

[Homebrew][2] formulae for my packages.


## Usage

```shell
$ brew tap arturoherrero/formulae
$ brew install <formula>
```

To update the brew package, use brew update first:

```shell
$ brew update
$ brew upgrade arturoherrero
```


## Available Formulae

- [bart][4]
- [gvar][3]
